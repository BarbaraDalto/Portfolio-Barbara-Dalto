#Portfólio - Bárbara Dalto

Site de portfólio pessoal desenvolvido para apresentar minha trajetória, formação e projetos na área de tecnologia.

##Objetivo do Projeto

O projeto foi desenvolvido como parte da minha formação em Engenharia de Software, com o objetivo de aplicar na prática conceitos de desenvolvimento web, organização de conteúdo, estruturação de páginas e criação de uma interface responsiva e intuitiva.

##Funcionalidades

- Navegação entre páginas (Sobre mim, Formação, Portfólio e Contato)
- Seção “Sobre mim” com apresentação pessoal e profissional
- Página de formação acadêmica e idiomas
- Exibição de projetos com descrição, funcionalidades e preview
- Links externos para projetos hospedados
- Página de contato com redes sociais e informações diretas
- Formulário com validação de campos via JavaScript
- Layout responsivo para diferentes dispositivos

##Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
