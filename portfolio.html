    Landing Page
    Corretora Andréia Souza

    Acesse o projeto

    Objetivo do Projeto: O projeto foi criado como exercício prático do Curso em Vídeo (HTML5 e CSS3), com foco em layout, tipografia, paleta de cores e responsividade, simulando um projeto real voltado para cliente final, onde simula uma Landing page institucional desenvolvida para apresentar os serviços da corretora Andréia Souza, especialista em negócios imobiliários na Praia dos Ingleses – SC.

    Funcionalidades:
  • Cabeçalho com identidade visual e navegação social
  • Botões de contato direto (WhatsApp, Facebook e Instagram)
  • Seção Hero com imagem de destaque e chamada estratégica
  • Seção “Quem sou eu” com apresentação profissional
  • Vídeo do YouTube incorporado
  • Bloco de contatos com e-mail, WhatsApp e localização
  • Rodapé com créditos e direitos autorais
  • Layout responsivo para dispositivos móveis

    • Tecnologias Utilizadas:
  • HTML5
  • CSS3
  • YouTube Embed para conteúdo multimídia

    Projeto
    Redes Sociais

    Acesse o projeto

    Objetivo do Projeto: Projeto visual desenvolvido em HTML5 e CSS3 que simula a navegação em um smartphone, exibindo diferentes páginas de redes sociais dentro de um iframe, visando demonstrar conceitos de layout, posicionamento, uso de iframes e interação visual, aplicados em um contexto prático.
    Este projeto foi criado como exercício prático do Curso em Vídeo (Gustavo Guanabara).

    Funcionalidades:
  • Simulação de tela de smartphone (iPhone)
  • Navegação interna entre páginas usando iframe
  • Botões laterais para redes sociais(Home, Instagram, Facebook, GitHub)
  • Layout centralizado com imagem de fundo fixa
  • Efeito hover nos ícones das redes sociais
  • Estrutura simples e intuitiva

    • Tecnologias Utilizadas:

  • HTML5
  • CSS3
  • Iframes

    Projeto Formulário HTML
    Controles Avançados

    Acesse o projeto

    Objetivo do Projeto:
    Este projeto foi criado como exercício prático do Curso em Vídeo (Gustavo Guanabara), a fim de desenvolver um formulário web utilizando HTML5 para praticar a criação e organização de campos de entrada de dados, permitindo a coleta estruturada de informações do usuário em uma página web.

    Funcionalidades:
  • Formulário de login com usuário e senha
  • Coleta de dados pessoais (nome, ano de nascimento, e-mail e telefone)
  • Cálculo automático da idade a partir do ano de nascimento
  • Seleção de estado através de lista suspensa
  • Sugestão de profissões utilizando datalist
  • Seleção de sexo com radio buttons
  • Upload de foto de perfil
  • Seleção de esportes favoritos com checkbox
  • Controle de nível de satisfação com range
  • Campo de mensagem com textarea
  • Mensagem de confirmação após envio do formulário

    • Tecnologias Utilizadas:

  • HTML5
  • CSS Interno
